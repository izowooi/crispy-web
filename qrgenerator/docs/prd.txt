# WiFi QR Code Generator - 요구명세서 (PRD)

## 1. 프로젝트 개요
WiFi 연결 정보를 입력하면 QR 코드를 생성하여 손쉽게 WiFi에 연결할 수 있도록 하는 웹 애플리케이션

## 2. 기술 스택
- 프레임워크: Next.js 14+ (App Router)
- 스타일링: Tailwind CSS
- QR 코드 생성: qrcode.react 또는 qr-code-styling (모던 디자인 지원)
- 배포: Vercel (권장)

## 3. 기능 요구사항

### 3.1 입력 필드
| 필드명 | 타입 | 필수 여부 | 설명 |
|--------|------|-----------|------|
| SSID (WiFi 이름) | text | 필수 | WiFi 네트워크 이름 |
| 비밀번호 | password | 필수 | WiFi 비밀번호 (표시/숨김 토글) |

### 3.2 보안 유형
- WPA/WPA2 고정 (가장 일반적인 보안 방식)

### 3.3 QR 코드 생성
- WiFi 연결 문자열 형식: WIFI:T:WPA;S:<SSID>;P:<password>;;
- 실시간 미리보기 (입력 시 즉시 QR 업데이트)

### 3.4 다운로드 기능
- PNG 형식 다운로드
- SVG 형식 다운로드 (선택)

### 3.5 디자인
- 모던하고 깔끔한 UI
- 반응형 디자인 (모바일/데스크톱)
- 다크모드 지원 (시스템 설정에 따라 자동 전환)

## 4. UI/UX 요구사항
- 단일 페이지 애플리케이션
- 직관적인 폼 레이아웃
- QR 코드 미리보기 영역
- 다운로드 버튼

## 5. 비기능 요구사항
- 모든 처리는 클라이언트 사이드에서 수행 (보안)
- 입력된 정보는 서버에 저장하지 않음
- 빠른 로딩 속도
